## 学习成本

据说Google的工程师第一次看到Python之后，进化了**2h**就完成了学习。实际上这是完全可能的，在具有其他计算机语言的开发经验和基础的逻辑思维之后，学习**Python**是非常简单的事情。一般来说，普通人学习Python在**15**天左右，大家可以利用寒暑假学习。
